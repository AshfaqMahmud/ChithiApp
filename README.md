# ChithiApp - Real-Time Chat App

A real-time chat application built with **Node.js**, **Express**, **Socket.IO**, and **MongoDB**. This app allows users to register, log in, send personal messages, and see a list of connected users.

---

## Features

- **User Authentication**:
  - Register and log in with a username and password.
  - JSON Web Tokens (JWT) for secure authentication.

- **Real-Time Messaging**:
  - Send and receive messages in real-time using **Socket.IO**.
  - Personal chat between two users.

- **Connected Users List**:
  - See a list of all connected users.
  - Select a user to start a personal chat.

- **Responsive UI**:
  - Clean and user-friendly interface.

---

## Technologies Used

- **Backend**:
  - Node.js
  - Express
  - Socket.IO
  - MongoDB (with Mongoose)
  - JSON Web Tokens (JWT) for authentication

- **Frontend**:
  - HTML
  - CSS
  - JavaScript (Vanilla JS)
